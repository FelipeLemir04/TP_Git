# Trabajo Practico Git y GitHub

## Objetivo
Aprender el uso basico de Git y GitHub mediante la creacion y versionado de un proyecto simple.

## Actividades realizadas

- Creacion de repositorio local.
- Creacion de tres commits descriptivos.
- Creacion de repositorio remoto en GitHub.
- Vinculacion entre repositorio local y remoto.
- Publicacion del proyecto en GitHub.

## Comandos utilizados

git init
git add .
git commit -m "mensaje"
git remote add origin URL
git push -u origin main
